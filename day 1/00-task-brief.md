# Task Brief (as given)

**Task:** LLM Comparison – Document Summarization

## Objective
Evaluate and compare the performance of different Large Language Models (LLMs) on a document summarization task.

## Task Description
Select the same article, research paper, or PDF document and generate summaries using at least three different LLMs (e.g., GPT, Gemini, Claude, Llama, Mistral, or Groq-hosted models). Analyze and compare the generated summaries to understand the strengths and weaknesses of each model.

## Requirements
- Use the same input document for all models.
- Use an identical prompt for each model to ensure a fair comparison.
- Test at least three different LLMs.
- Record the output generated by each model.

## Evaluation Criteria
1. **Summary Quality** — Is the summary well-structured and easy to understand?
2. **Accuracy** — Does the summary correctly represent the information in the source document?
3. **Conciseness** — Does the model avoid unnecessary details while preserving important information?
4. **Hallucinations** — Does the summary include information that is not present in the original document?

## Deliverables
- Original document used for testing.
- Prompt used for all models.
- Summaries generated by each model.
- Comparison table containing scores or observations for each evaluation metric.
- A short conclusion identifying which model performed best and why.

## Acceptance Criteria
- At least three LLMs have been evaluated.
- The same document and prompt were used for all models.
- Comparison results are documented clearly.
- Final observations and recommendations are provided.
